## Requirements

- Node.js 18+
- npm

## System tools (install separately)
- lighthouse: `npm install -g lighthouse`
- jq 1.6+: `winget install jqlang.jq` (Windows)
- curl 7.x+ (comes with Git Bash)
- openssl 1.1+ (comes with Git Bash)

## Run

node src/index.js https://example.com
